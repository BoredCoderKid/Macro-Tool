# Macro Tool

## Description
Simple macro tool which fetches boring stuff like mouse pos or pixel color
## Dependencies
* Python
* customtkinter

## Installing
Just install the python file and you're done!

---

## How to run the program
Run the program using the `.pyw` Python file.
